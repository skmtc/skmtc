# SkipOperations

Configuration for skipping operation generation by generator type.

Maps generator keys to [SkipPaths](./skippaths.mdx) configurations for excluding
specific operations from generation.


## Example

```typescript
const skipOperations: SkipOperations = {
  'api-client': {
    '/internal/**': ['*'],
    '/admin': ['delete']
  }
};
```
