---
title: OasSchema
description: Union of OpenAPI schema objects
---

OasSchema is not an object itself, but a union of OpenAPI schema objects. It can be one of

- [OasArray](/open-api/oas-array)
- [OasBoolean](/open-api/oas-boolean)
- [OasInteger](/open-api/oas-integer)
- [OasNumber](/open-api/oas-number)
- [OasObject](/open-api/oas-object)
- [OasString](/open-api/oas-string)

as well as

- [OasUnknown](/open-api/oas-unknown)
- [OasUnion](/open-api/oas-union)
- [OasIntersection](/open-api/oas-intersection)
