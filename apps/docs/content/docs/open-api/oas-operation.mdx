description: Object representing an OpenAPI operation

OasOperation is an object representing an OpenAPI operation.

In addition to standard OpenAPI Operation object properties, it contains the api operation `path` and `method`.

## Properties

| Property        | Type                | Description  |
| --------------- |-------------------- | ------------ |
| **oasType**     | `'operation'`            | Static value representing the OpenAPI operation type. |
| **path**        | `string`            | Operation path |
| **method**      | `string`            | Lowercase HTTP method |
| **operationId** | `string \| undefined` | Optional operation id |
| **description** | `string \| undefined` | Optional description |
| **tags**        | `string[] \| undefined` | Optional list of string tags which can be used to group operations |
| **parameters**  | `(OasParameter \| OasRef<'parameter'>)[] \| undefined`    | Optional array of path, **query**, **header** or **cookie** [OasParameter](/open-api/oas-parameter) or references to them.  |
| **requestBody** | `OasRequestBody \| OasRef<'requestBody'> \| undefined` | Optional [OasRequestBody](/open-api/oas-request-body) or reference to it. |
| **responses**   | `Record<string, OasResponse \| OasRef<'response'>>` | Record of [OasResponse](/open-api/oas-response) or references to them by HTTP status code. |
| **pathItem**    | `OasPathItem` | [OasPathItem](/open-api/oas-path-item) with meta data about the path. |
| **deprecated**  | `boolean \| undefined` | Optional flag marking the operation as deprecated. |
| **extensionFields**  | `Record<string, unknown> \| undefined` | Optional map of custom fields defined by OpenAPI extensions. Keys are strings beginning with `x-` and values are of any valid JSON type. |


## Methods

### toSuccessResponse

Returns the first response with a status code of 2xx or 'default' if no such response exists.

```js
operation.toSuccessResponse()
```

**Return value**

An [OasResponse](/open-api/oas-response) or [OasRef](/open-api/oas-ref) object.

### toRequestBody

Get the request body schema for the given media type.

```js
operation.toRequestBody( mediaType = 'application/json')
```

<details className="mt-5">
<summary>**Parameters**</summary>

| Property        | Type                | Description  |
| --------------- |-------------------- | ------------ |
| **mediaType**   | `string`            | Media type of the request body. Defaults to `application/json`. |

</details>

**Return value**

An [OasSchema](/open-api/oas-schema) or [OasRef](/open-api/oas-ref) object.

### toParams

Get all resolved parameters for the operation.

```js
operation.toParams( filter )
```

<details className="mt-5">
<summary>**Parameters**</summary>

| Property        | Type                | Description  |
| --------------- |-------------------- | ------------ |
| **filter**      | `('path' | 'header' | 'query' | 'cookie')[] \| undefined` | Optional array of parameter locations to filter by. Returns all parameters when **filter** is not provided. |

</details>

**Return value**

An array of [OasParameter](/open-api/oas-parameter) objects.

### toParametersObject

Get all resolved parameters for the operation as an [OasObject](/open-api/oas-object).

```js
operation.toParametersObject( filter )
```

<details className="mt-5">
<summary>**Parameters**</summary>

| Property        | Type                | Description  |
| --------------- |-------------------- | ------------ |
| **filter**      | `("path" | "header" | "query" | "cookie")[] \| undefined` | Optional array of parameter locations to filter by. Returns all parameters when **filter** is not provided. |
</details>

**Return value**

An [OasObject](/open-api/oas-object) containing all resolved parameters for the operation.
