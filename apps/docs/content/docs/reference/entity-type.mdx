---
title: EntityType
description: Object representing a variable or type
---

EntityType is an object representing a variable or type.

It is primarily used as a property of the [Identifier](/identifier) object to generate a `const` or `type` keyword when the Identifier is rendered by a [Definition](/definition).

## Properties

| Property        | Type                | Description  |
| --------------- |-------------------- | ------------ |
| **type**        | `'variable' \| 'type'`            | Type of entity |
