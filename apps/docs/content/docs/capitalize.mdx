# capitalize

Capitalizes the first character of a string.


**@param** str - The string to capitalize

**@returns** The string with the first character uppercased


## Example

```typescript
capitalize('hello world'); // 'Hello world'
capitalize('API'); // 'API'
capitalize(''); // ''
```

## Variable Definition

```typescript
const capitalize: (str: string) =\> string
```
